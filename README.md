# BBDown_GUI
BBDown的图形化版本，亦保留有命令行

## 特性

- [x] 记忆之前的参数

## 下载
[![img](https://img.shields.io/github/release/1299172402/BBDown_GUI?label=版本)](https://github.com/1299172402/BBDown_GUI/releases)

## 从源码运行
```
pip install -r requirements.txt
python BBDown_GUI.py
```
